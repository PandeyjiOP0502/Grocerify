# Grocerify

A modern grocery shopping website built with React + Vite.

## Features

- 🛒 Full shopping cart with add/remove/quantity controls
- 🔍 Product search and category filtering
- 📱 Fully responsive design
- 🎨 Modern UI with Outfit font and green color scheme
- 📦 Product detail pages with related products
- 💰 Order summary with price breakdown

## Tech Stack

- React 19
- React Router v7
- Vite 8
- CSS3 (Vanilla)

## Getting Started

```bash
cd client
npm install
npm run dev
```

## Pages

- **Home** - Hero section, categories, best sellers, features
- **Products** - Filterable product grid with sidebar
- **Product Detail** - Full product info with add to cart
- **Cart** - Shopping cart with order summary
- **Contact** - Contact form and info
